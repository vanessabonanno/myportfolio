Website redesign

The redesign of the SMC company website aims to create a product that could facilitate usability, comply with accessibility standards and which would follow the latest design trends.

This project was started in 2022 by my colleagues, who set the “why”, the main goals the marketing office wanted to accomplish through the redesign. The marketing office’s need was to freshen up, riding the wave of the latest design trends, and also stick to the visual identity of the already existing SMC's analog graphics.

I was introduced to a partially finished UI kit, which already included the colour palette, styles, icons, shadows, typography, buttons, and some patterns which could be used in the design.

My first task was to create some missing components such as buttons, navbar, footer, cards, newsletter. Then I started sketching the homepage with some variants and eventually I designed all the pages.

In the end I focused on the composition of the Design system and the detailed description of its use. It represents a guideline fot the developers, helping and simplifying the transition fro the Figma board to the web.
