SMC website redesign

Role: UI Designer, responsive design specialist

Scope: Material Design-driven UI overhaul, bridging analog brand identity with modern digital trends.

Context

The redesign of the SMC corporate website was the first project I worked on as a UI designer. The main goal was to evolve the brand's digital presence towards contemporary usability and accessibility standards. The challenge was to align the interface with the digital standards, while maintaining strict continuity with the company’s established visual identity, ensuring full alignment and compliance with WCAG 2.1 accessibility guidelines.

Joining the project during an intermediate phase, my primary task was to transform a partially defined UI kit into a comprehensive, functional, and robust design system, which followed Google’s Material Design principles. Thus, I developed missing core components — including navbars, footers, newsletter templates, and complex "annotation cards" — ensuring that every element balanced scalability with aesthetic precision.
